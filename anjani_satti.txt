git clone--retrive an entire repository from a hosted location via URL.
git push origin--to upload local repository content to a remote repository.
cd--to change the directory.
git branch--create a new branch at the current commit.
git checkout--switch to another branch and check it out into your working directory
git merge[branch]--merge the speicified branch's history into the current one.
git checkout -b --to create and switch to the new branch
git add[file]--add a file as it looks now into your next commit(stage).
git commit -m "[descriptive message]"--commit your staged content as a new commit snapshot.
touch [filename]--adding the file to the current index.
git mv--move or rename a file or directory.
